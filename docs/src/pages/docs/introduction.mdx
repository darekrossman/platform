# Introduction

ESS is a set of utilities and helpers for creating token-based design systems. Together, the `ess` and `ui` packages provide a robust styling solution for React apps by leveraging the power of [Emotion](https://emotion.sh/docs/introduction) and [styled-system](https://styled-system.com/).

### Features

- Includes `@emotion/core` for `css` prop goodness
- Flexible, with an eye towards rapid UI development
- Fully themeable with token-based prop APIs
- Easy responsive styles with prop arrays
